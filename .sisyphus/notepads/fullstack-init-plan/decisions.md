# Architectural Decisions - Full-Stack Init Plan

This file tracks key architectural and technical decisions made during execution.

---

## Initial Plan Decisions

- **Database**: MongoDB with Mongoose ODM
- **Backend Architecture**: Layer Architecture (Domain, Application, Infrastructure, Presentation)
- **Frontend Router**: Next.js App Router (modern approach)
- **Styling**: Tailwind CSS
- **Package Manager**: npm
- **Node Version**: 20+
- **Testing**: Playwright for both E2E and API tests
- **CI/CD**: GitHub Actions with staging/production environments
